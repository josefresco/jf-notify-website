Common Questions

Still not sure? If your question isn't answered here, feel free to get in touch.

How long does setup actually take?
About 5 minutes from start to finish. Create a Telegram bot with @BotFather (2 minutes), install and activate the plugin (1 minute), paste your Bot Token and Chat ID into the settings page and save (1 minute), then submit a test form to confirm it's working (1 minute). That's it.
Does it work with all Gravity Forms?
Yes. JF Notify hooks into Gravity Forms at the submission level, so it automatically captures every form on your site — contact forms, quote requests, registration forms, order forms, anything. No per-form configuration needed.
What if Telegram has an outage?
Telegram is extremely reliable with near-zero downtime. In the highly unlikely event of an outage, Gravity Forms still stores all entries in your WordPress database as normal — no data is lost. You can also keep WordPress email notifications enabled as a secondary backup if you prefer.
Is this a subscription? Any recurring fees?
No subscription, no annual renewal, no recurring fees of any kind. You pay $29 once and the plugin is yours indefinitely. Updates are included and there is no expiration on your license.
Can I use it on multiple websites?
Yes — one purchase, unlimited sites. There is no per-site licensing restriction. Whether you manage 1 site or 50, one payment covers all of them.
